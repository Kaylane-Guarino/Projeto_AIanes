Estrutura dos comandos HTML5

Tags para estrutura do site HTML5 header - representa a seção do cabeçalo do site footer - representa a seção do rodapé no site nav- representa a seção do menu de navegação do site, pórem deve ser utilizada apenas no menu relevante do site section- representa as seções de conteudo do site (assuntos) article- representa um conteudo de terceiros,que podemos coloar no site(Citação de um autor,blog,forum,area de reportagem que vão vir de outros sites) aside- representa uma area de propaganda de outro site (divulgar produtos de terceiros,anuncios) figure - indica que ali terá uma imagem (img), que é relevante para o assunto do site figcaption - perimte colocar uma legenda na imagem(não é obrigatório) video - permite colocar um video canvas - permite criar um desenho (grafico) audio- permite colocar um audio
